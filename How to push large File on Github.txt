1. Install Git LFS

Download and install from:
https://git-lfs.com/

Then run:

git lfs install

2. Tell Git LFS to track large file types

For example, to track all .mp4 files:

git lfs track "*.mp4"


This creates a .gitattributes file.

3. Add files
git add .

4. Commit changes
git commit -m "Enable Git LFS for videos"

5. Push to GitHub
git push
